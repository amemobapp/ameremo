# Google Business Profile API 設定依頼

## 📋 目的
アメモバ店舗の口コミデータを全件取得し、長期的な評判管理システムを構築するため、Google Business Profile APIの設定をお願いします。

## 🎯 期待される効果
- ✅ **全口コミ取得**: 現在5件しか取得できない口コミを全件取得可能に
- ✅ **履歴データ**: 過去の口コミ履歴を分析し、評判推移を把握
- ✅ **自動化**: 手作業でのデータ収集から解放され、定期的なモニタリングを実現

## 🔧 必要な作業（IT担当者向け）

### 1. Google Business Profileの確認
```
確認対象店舗：
- アメモバ買取 上野店
- アメモバ買取 秋葉原店
- アメモバ買取 柏店
- アメモバ買取 名古屋大須店
- アメモバ買取 新宿東南口店
- アメモバ買取 大宮マルイ店
```

### 2. Google Cloud Consoleでの設定
1. **プロジェクト作成**: 新規または既存のGCPプロジェクト
2. **API有効化**: 「Business Profile API」を有効化
3. **認証情報作成**: OAuth 2.0 クライアントIDを作成

### 3. OAuth 2.0 設定
- **承認済みリダイレクトURI**: `http://localhost:3000/api/auth/callback`
- **スコープ**: `https://www.googleapis.com/auth/business.manage`

### 4. 権限確認
- 各店舗の「管理者」または「オーナー」権限があること
- Business Accountの管理権限があること
