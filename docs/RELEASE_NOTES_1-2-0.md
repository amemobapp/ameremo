# リリースノート

アメレモン更新のお知らせ
バージョン：v1.2.0
公開日：2026/04/14
概要：**アメレモ（AmeReMo）** のデータ整合性修正とダッシュボードUI改善リリースです。
https://ameremo.vercel.app/

## 更新内容

### バグの修正
1）アメモバ 新宿東南口店の `Place ID` を正しい値へ訂正し、Google口コミを正しく取得できるようにした。

### 期間別・評価別テーブルの改善：
2）店舗名Prefix変更（Am→Ⓐ、Sk→Ⓢ）し、よりスマートになった。
3）店舗名をハイパーリンクにし、Google Mapを直接開けるようになった。
4）ヘッダーで1店舗を選択したとき、テーブルではその店舗のみ表示し、0件になる他店舗を非表示とし、見た目がよりクリーンになった。
5）`店舗名`列の右に`件数合計`列を追加したことで、合計件数を簡単に把握できるようになった。
6）`店舗名`と`合計件数`に並べ替えボタンを追加し、クリックで昇順・降順を切り替えられるようになった。
7）チャートと口コミ一覧のURLを変えた
8）サービス名を「アメレモン」に変更した（🍬と🍋、甘い口コミと酸っぱい口コミの意味です）
