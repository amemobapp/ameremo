# リリースノート

## v1.1.1

**アメレモ（AmeReMo）** の軽微な改善リリースです。

---

### 概要

店舗選択ドロップダウンの表示を改善し、選択中の店舗が分かりやすくなりました。

---

### v1.1.1 の変更

#### スマホ向け UI

| 変更内容 |
|----------|
| **店舗選択**ドロップダウンのボタンで、選択状態を表示するように変更 |
| 特定の店舗が **1件だけ** 選択されているときは、ボタンに **その店舗名**（短縮名）を表示 |
| **複数店舗** 選択時は「店舗（N件）」と表示。すべて選択時は従来どおり「店舗を選択」 |

---

### 達成したこと（v1.1.1）

- [x] 店舗選択ボタンに選択中の店舗名／件数を表示
