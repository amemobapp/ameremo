# リリースノート

## v1.1.0

**アメレモ（AmeReMo）** の UI・文言・フォントの改善リリースです。

---

### 概要

v1.0 の機能を維持しつつ、サイト名の統一、スマホ向けナビ・モーダルの見直し、グラフ凡例・テーブル表示の調整、フォントの統一などを行いました。

---

### v1.1.0 の主な変更

#### サイト名・ナビゲーション

| 変更内容 |
|----------|
| サイトタイトルを **「アメレモ」** に統一（ブラウザタブ・ヘッダー表記） |
| サイトタイトル（アメレモ）をクリックすると **ルート（/）に戻る** ように変更 |
| タイトル右に **バージョンバッジ**（v1.1.0）を上付き風で表示。コンパクトな padding・フォント・上下均等の余白で表示 |

#### スマホ向け UI

| 変更内容 |
|----------|
| 店舗選択を **メニューボタン＋ドロップダウン** に変更。ボタンは「店舗を選択 ▼」と表示し、タップで直下にパネルを表示（右端まで幅いっぱい） |
| ブランド・店舗を **1行で表示**（すべて / アメ / サク / 店舗を選択） |
| **フィルターボタン**（じょうごアイコン）のモーダルを、画面中央ではなく **ボタン直下のドロップダウン** で表示するように変更 |
| 口コミタブ選択時の下のインジケーター（丸）を削除 |
| **店舗を選択**モーダルの閉じるボタン（×）を大きくし、タップしやすくした |
| **日付・フィルター**モーダルの閉じるボタンを **SVG アイコン＋丸ボタン** に変更 |

#### ダッシュボード・サマリ

| 変更内容 |
|----------|
| **指定期間口コミ件数**・**平均評価** のカードを縦にコンパクトにし、**スマホでも横並び**（2列）で表示 |
| 見出しを「口コミ投稿件数」→ **「指定期間口コミ件数」** に変更 |
| 注釈を「期間内の総投稿件数」「※取得は店舗・期間ごとに最新5件までです。」→ **「現在は直近の分のみ取得しています。」** に変更 |
| 指定期間口コミ件数・平均評価の **フォントを約1.5倍** に拡大（見出し・数値） |
| 表示順を **時系列推移 → 期間別 → 評価別** に変更（期間別を評価別の上に移動） |

#### 時系列グラフ

| 変更内容 |
|----------|
| 凡例を **■口コミ件数**・**●平均評価** の文字＋色（青・緑）のみの 1 行表示に変更（アイコンは 1 文字サイズ） |
| 凡例の下にあった補足テキスト（「青色の棒グラフ：…」「緑色の折れ線グラフ：…」）を削除 |
| 右軸（平均評価）の目盛りを **0, 1, 2, 3, 4, 5** に固定。最大スケールは 6 のままだが **6 は表示しない** |

#### テーブル

| 変更内容 |
|----------|
| **評価別**テーブルの行の高さ・セル余白を **期間別**テーブルと同一に調整 |
| **期間別**テーブルの店舗名列の **右側にシャドー** を付与（固定列の境目を強調） |
| テーブルの店舗名セルに **GlowSansSC Compressed Medium** を適用（既存のまま。v1.1 ではサイト全体フォントと統一） |
| 評価別テーブル内の星の中の数字の縦位置を少し下に調整 |

#### フォント

| 変更内容 |
|----------|
| サイト全体の本文を **GlowSansSC Compressed Medium** に統一 |
| 注釈・補足テキスト（期間説明、凡例、件数表示、ページネーション、レビュー著者・日付など）を **GlowSansSC Compressed Light** に統一 |

#### フィルターモーダル

| 変更内容 |
|----------|
| **集計粒度**（日別 / 週別 / 月別）をドロップダウンから **ボタン式・横並び 1 行** に変更 |
| **口コミの並び順**（新しい順 / 古い順 / 評価高い順 / 評価低い順）をドロップダウンから **ボタン式** に変更 |

---

### 達成したこと（v1.1.0）

- [x] サイト名「アメレモ」への統一とタイトルクリックでルートへ戻る
- [x] スマホで店舗を「店舗を選択」ドロップダウンに集約（右端まで幅いっぱい）
- [x] スマホでブランド・店舗を 1 行表示（すべて / アメ / サク / 店舗を選択）
- [x] フィルターをボタン直下のドロップダウンで表示
- [x] サマリカードの文言変更（指定期間口コミ件数・直近取得の注釈）とフォント拡大
- [x] サマリを常に横並び・縦コンパクトに
- [x] 期間別を評価別の上に配置
- [x] 時系列グラフの凡例を ■/● ＋ 色付き文字の 1 行に簡略化、右軸を 0〜5 刻みに
- [x] 評価別・期間別テーブルの行高・余白の統一、期間別店舗名列に右シャドー
- [x] サイト全体を Medium、注釈を Light フォントで統一
- [x] タイトル右にバージョンバッジ（v1.1.0）を表示
- [x] 店舗を選択・日付・フィルターの閉じるボタンを大きく／SVG・丸ボタンに変更
- [x] 集計粒度・口コミの並び順をボタン式に変更

### 技術スタック（v1.1.0）

v1.0 と同様（Next.js, React, TypeScript, Tailwind CSS, PostgreSQL + Prisma, Google Places API）。  
フォントは `next/font/local` で GlowSansSC-Compressed-Medium.otf / Light.otf を読み込み、CSS 変数と Tailwind の `font-sans`・`.font-annotation` で適用。
